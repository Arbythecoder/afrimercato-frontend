## VIDEO PRODUCTION NOTES

### Key UI Elements to Simulate
1. **Navigation Bar** - Logo, Products, Stores, Cart icon with badge
2. **Store Cards** - Logo, name, category, rating
3. **Product Cards** - Image, name, price, Add to Cart button
4. **Cart Page** - Items list, quantity controls, checkout button
5. **Checkout Flow** - 3-step progress bar, forms, order summary
6. **Order Tracking** - Timeline, map with moving rider dot

### Animation Suggestions
- Product "flying" to cart icon
- Cart badge incrementing
- Checkmark celebration on order confirmation
- Rider dot moving on map
- Timeline steps lighting up in sequence

### Color Palette
- Primary Green: #22C55E (buttons, success states)
- Secondary Gold: #EAB308 (highlights, ratings)
- Background: White/Light Gray
- Text: Dark Gray (#111827)

### Transitions
- Smooth fades between pages
- Slide transitions for checkout steps
- Zoom effects for callouts
- Bounce animations for buttons
