## PRODUCTION NOTES FOR ADDITIONAL VIDEOS

### These videos are shorter and can use:
- Simpler animations
- Fewer callouts
- Same visual style as main videos
- Consistent voiceover tone

### Priority Order:
1. **Vendor Video** (most important - drives platform growth)
2. **Customer Video** (essential for user adoption)
3. **Rider Video** (important for operations)
4. **Picker Video** (internal training)
5. **Admin Video** (internal use only)

### Alternative: Single Combined Video
Instead of 5 separate videos, consider:
- **"Afrimercato Platform Overview"** (8-10 minutes)
- Cover all roles in chapters
- Use YouTube chapters for navigation
- Viewers can skip to their role
